# KaomojiClash ゲームマニュアル

KaomojiClash は、顔文字を組み立てて戦うビルド主導型トップダウンアクションです。
本ドキュメントはプレイ体験とゲームシステムを中心に整理しています。

## 1. ゲームの核

- 強さはレベルよりも「顔文字パーツ構成」で決まる
- 引っ張って発射するシンプル操作で高い戦略性を作る
- 記号収集 -> ビルド再構成 -> 再挑戦のループで成長する

## 2. 基本ループ

1. エリアを選ぶ
2. Wave 戦闘を開始する
3. 敵を撃破してドロップを得る
4. 顔文字パーツを組み替える
5. 難易度を上げて次の挑戦へ進む

## 3. 戦闘

### プレイヤー操作

- Drag & Release でキャラクターを射出
- 壁反射を使って位置取りと追撃を行う
- 条件付きスキルで一時的に性能を強化
- アリーナ設置物を活用して戦況を制御

### 戦闘ステータス

| ステータス | 役割 |
|---|---|
| Speed | 発射速度・機動力に影響 |
| Power | 衝突ダメージに影響 |
| Guard | 被ダメージ軽減に影響 |
| Stamina | 最大体力に影響 |

各ステータスは装備パーツの合計値から決定されます。

## 4. 顔文字ビルド

### パーツスロット

| パーツ | スロット | 影響倍率 |
|---|---|---|
| Mouth | 中央 | x1.0 |
| Eyes | 左右 | x0.5 |
| Hands | 装飾 | x0.3 |
| Decoration First | 追加 | x0.1 |
| Decoration Second | 追加 | x0.1 |

- パーツは重複装備可能
- 戦闘報酬で新規パーツを解放
- 経験値により個別パーツが成長

### 成長率タイプ

最終成長値は次式で決定されます。

Final Growth = Base Growth x Growth Rate

| タイプ | 倍率 |
|---|---|
| VeryLow | 0.85 |
| Low | 0.92 |
| Normal | 1.00 |
| High | 1.15 |
| VeryHigh | 1.20 |

## 5. 難易度

| 難易度 | 敵倍率 | 経験値倍率 | 色 |
|---|---|---|---|
| Easy | x0.95 | x1.00 | 緑 |
| Normal | x1.00 | x1.25 | 黄 |
| Hard | x1.15 | x1.50 | オレンジ |
| Extreme | x1.25 | x1.75 | 赤 |

低難易度でビルドを整え、高難易度で報酬効率と挑戦性を高める設計です。

## 6. アリーナアイテム

- Rebound Pad
- Healing Pad
- Acceleration Area
- Medikit

Medikit は接触時に最大体力割合で即時回復し、使用後はクールタイムで再使用待ちになります。

## 7. スキル

タグベースで管理され、同一スキル重複時はレベルと効果が上昇します。

例:
- Shunsoku
- Kinniku Zokyo
- Teppeki
- Rebound Boost

## 8. オーディオと演出

### BGM 遷移

- シーンとフェーズに応じて BGM を動的切替
- フェードアウト -> 切替 -> フェードインの流れで違和感を軽減
- 多重切替時は進行中コルーチンを中断して再開

### バトル演出

- ダメージテキスト表示
- Hit Stop / Hit Zoom
- Cinemachine 連動のカメラシェイク
