# KaomojiClash ストーリー背景

## 1. 世界観

遠い未来、文字は単なる記録手段ではなく、感情と意志を宿すエネルギー体へと変化した。

人々はその力を Kaomoji と呼び、記号の並びで人格・能力・記憶の輪郭を形作る技術を発展させた。
顔文字は遊びではなく、戦場で実体化する戦術フレームとなる。

## 2. 文明の構造

世界は複数の文化圏レイヤーに分断されている。
各文化圏は独自の記号体系を持ち、同じ意味でも異なる表現と力学を生む。

- Low Layer: 基礎記号が中心、安定だが拡張性が低い
- Mid Layer: 複合記号が増え、戦術の幅が広い
- High Layer: 希少記号が支配し、強力だが制御が難しい

プレイヤーは層を越えて記号を収集し、自分だけの顔文字を再構築していく。

## 3. 対立

中心都市 Lexicon の深層で、未知のノイズ領域 Null が拡大している。
Null は記号の意味を歪め、人格を持つ顔文字ユニットを暴走させる現象。

主人公は失われた辞書庫を追う探索者として、各エリアで暴走体を鎮静化しながら
崩壊した記号秩序の原因に迫る。

## 4. 主人公の立場

主人公は一つの完成体ではなく、装備した記号パーツによって性質が変わる可変型フレーム。

- 口パーツ: 中心的な行動特性を決定
- 目パーツ: 反応速度や感知を補強
- 手パーツ: 補助特性を付与
- 装飾: 微調整と相乗効果の起点

成長とはレベル上昇ではなく、意味を持つ記号の組み合わせを更新し続けること。

## 5. 物語進行の軸

1. エリア侵入
2. Wave でのノイズ鎮圧
3. 記号パーツ回収
4. ビルド再構築
5. より高層の文化圏へ到達

この流れがゲームプレイと物語進行を一致させ、プレイヤー体験を一本化する。

## 6. テーマ

- 強さは個体性能ではなく構成に宿る
- 失われた意味を再編集して未来を作る
- 違う文化圏の記号をつなぎ、新しい表現を生む

## 7. 将来の拡張フック

- Trophy を用いた対戦マッチメイキング
- 文化圏ごとの記号ボーナス連携
- Null 由来の高リスク高報酬ルール
